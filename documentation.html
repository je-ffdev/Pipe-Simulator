Simulator Documentation

Overview

This web application is designed to perform various calculations related to fluid mechanics, such as calculating the average velocity in a pipe using Poiseuille's equation, estimating parameters, calculating Reynolds number, and computing frictional head loss using the Darcy-Weisbach equation. Additionally, the application allows for saving and loading of parameters, unit conversion between metric and imperial systems, and selection between different fluid mechanics models.

Functionalities

1. Save Webpage Content Functionality: Save the entire content of the webpage to a file. Code: saveFile() Usage: This function captures the entire HTML content of the webpage and saves it as a .balls file.

2. Unit Conversion Functionality: Toggle between metric and imperial units. Code: toggleUnits(), setUnits(), setMetricValues(), convertToImperial() Usage: Converts input fields and unit labels between metric and imperial units based on the selected unit system.

3. Form Selection Based on Pipe Type Functionality: Display specific forms based on selected pipe type. Code: Event listener for #type-dropdown Usage: Shows/hides forms for circular, square, or channel pipe types.

4. Model Form Selection Functionality: Display specific forms based on selected fluid mechanics model. Code: Event listener for #model-option Usage: Shows/hides forms for Darcy-Weisbach or Bernoulli model calculations.

5. Save and Load Parameters Functionality: Save current input parameters to a file and load them back. Code: saveParametersToFile(), loadParametersFromFile() Usage: Stores input parameters in a .txt file and reloads them as needed.

6. Average Velocity Calculation Functionality: Calculate average velocity using Poiseuille's equation. Code: Event listener for #calculate-vavg Usage: Computes the average velocity based on pressure difference, pipe diameter, fluid viscosity, and pipe length.

7. Parameter Estimation Functionality: Estimate area and volumetric flow rate based on pipe diameter and average velocity. Code: Event listener for #estimate-parameters Usage: Computes and displays the area and flow rate, and displays additional parameter sections.

8. Reynolds Number Calculation Functionality: Calculate Reynolds number and determine flow regime. Code: Event listener for #calculate-btn, calculateTurbulentFrictionFactor() Usage: Computes Reynolds number, identifies flow regime, and calculates friction factor for turbulent flow.

9. Frictional Head Loss Calculation Functionality: Calculate frictional head loss using Darcy-Weisbach equation. Code: Event listener for #darcy-btn Usage: Computes head loss based on friction factor, pipe length, fluid velocity, and pipe diameter.

10. Bernoulli Equation Calculation Functionality: Calculate unknown parameters using Bernoulli's equation for different cases.

Event listeners for Bernoulli form and case selection Usage: Computes unknown values for pressure, velocity, or elevation in Bernoulli's equation.

Limitations

Transitional Flow

The application does not handle calculations for the transitional flow regime (Reynolds number between 2300 and 4000). An alert is shown to the user indicating the complexity.

Friction Factor Calculation

For turbulent flow, the friction factor calculation using the Colebrook equation may not converge within the set number of iterations. If convergence fails, an alert is shown

Input Validation

The application performs basic input validation, checking for negative or empty values. More comprehensive validation may be needed for specific use cases

Unit Conversion Assumptions

Assumes standard unit conversion factors. Users need to ensure input values are appropriate for the selected unit system.

Hardcoded Constants

Some constants, like gravitational acceleration (9.81 m/sÂ²), are hardcoded and may need adjustments based on specific requirements.

Form Dependency

Some calculations depend on prior inputs from other forms (e.g., Reynolds number calculation requires inputs from the average velocity form).

Usage Instructions

Select Pipe Type

Use the dropdown to select the type of pipe (circular, square, channel).

Select Model

Use the dropdown to select the fluid mechanics model (Darcy-Weisbach, Bernoulli)

Fill Input Fields

Enter the required values in the respective fields.

Calculate Average Velocity

Click the "Calculate Average Velocity" button to calculate the average velocity.

Estimate Parameters

Click the "Estimate Parameters" button to compute area and flow rate.

Calculate Reynolds Number

Click the "Calculate Reynolds Number" button to compute the Reynolds number and flow regime.

Calculate Head Loss

Click the "Calculate Head Loss" button to compute the frictional head loss.

Save/Load Parameters

Click the "Save/Load Parameters" button to save/load the parameters.

Toggle Units

Use the unit toggle to switch between metric and imperial units.

Calculate Bernoulli

For Bernoulli's equation, select the case and enter values, then click the "Calculate" button.
